# Контекст дипломного проекта

## Тема
Веб-приложение для организации и продвижения фестивалей и культурных мероприятий в малом городе.

## Цель
Разработка веб-приложения, обеспечивающего централизованную автоматизацию и информационную поддержку процессов планирования, организации, продвижения и посещения фестивалей и культурных мероприятий в малом городе.

## Задачи
- разработка архитектуры системы;
- проектирование базы данных;
- разработка серверной части;
- разработка клиентской части;
- реализация регистрации пользователей и ролей;
- реализация управления мероприятиями;
- реализация записи на мероприятия;
- реализация системы отзывов и избранного.

## Роли пользователей
- Житель (пользователь)
- Организатор
- Администратор

## Технологический стек
- Backend: Java 17, Spring Boot
- Frontend: React (Vite)
- Database: PostgreSQL
- ORM: Hibernate / JPA
- Безопасность: Spring Security + JWT

## Основной функционал
- регистрация и авторизация пользователей;
- просмотр афиши мероприятий;
- фильтрация мероприятий;
- просмотр карточки мероприятия;
- запись на сеансы;
- генерация QR-токена для регистрации;
- добавление в избранное;
- оставление отзывов;
- управление мероприятиями организатором;
- модерация контента администратором.
